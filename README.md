# 📋 Taskly — Task Manager App

A clean and bold task manager web app built with Flask and SQLite.

## 🚀 Features
- ➕ Add new tasks
- 👁️ View all tasks
- ⏳ Track pending tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 🔄 Auto status tracking

## 🛠️ Built With
- Python
- Flask
- SQLite
- HTML/CSS
- Jinja2

## ⚙️ How to Run
1. Clone the repo
   git clone https://github.com/yourusername/taskly.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   python to-do.py

4. Open in browser
   http://127.0.0.1:5000
